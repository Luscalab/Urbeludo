# **App Name**: UrbeLudo

## Core Features:

- Webcam-based Object Detection: Identifies urban-like architectural elements (e.g., lines, steps, walls) from a live webcam feed using on-device (browser-based) AI, adapting the original mobile vision to a Next.js web environment.
- Dynamic Challenge Generation Tool: The AI tool analyzes detected elements and user interaction to dynamically propose unique psychomotor challenges that stimulate balance, muscle tone, laterality, and spatial structuring, similar to a physical urban playground.
- Interactive Challenge Overlay: Visually presents challenges by overlaying instructions and highlights directly onto the live webcam stream, guiding the user in real-time within the urban environment.
- Challenge Completion Mechanism: Allows users to confirm completion of proposed challenges through a simple interface, enabling progression within the experience and providing immediate feedback.
- Local Progress Tracking: Stores user performance metrics, challenge streaks, and history locally within the browser to support personalized challenge progression and maintain user engagement, adhering to the privacy-first philosophy.
- Minimalist UI for Urban Interaction: Provides a clean, intuitive, and distraction-free user interface focused on enhancing urban exploration and psychomotor challenge engagement, maintaining performance and scalability.

## Style Guidelines:

- The primary color, '#33993D' (a muted, earthy green), evokes connection to active outdoor movement and grounded functionality, contrasting against a light background.
- The background color, '#F2FAF2' (a very light, subtly green-tinted white), supports the light color scheme and provides a clean, unobtrusive canvas for urban exploration without visual distractions.
- The accent color, '#99E630' (a vibrant yellow-green), provides high contrast for call-to-action elements and interactive challenge highlights, visually drawing attention without being overly bold.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and objective aesthetic, ensuring readability and supporting the app's minimalist and functional design philosophy.
- Use performance-optimized vector components (SVG) for all icons, favoring clean lines and universal symbolism to ensure clarity and scalability for challenge visualization and navigation.
- Employ a focused and uncluttered layout that prioritizes the live webcam feed and challenge overlay. Key interactive elements are presented contextually to facilitate intuitive urban interaction.
- Implement subtle, high-performance animations, such as smooth transitions for challenge changes or feedback, to enhance the user experience without introducing latency or distraction.
